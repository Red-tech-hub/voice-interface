# Company

**Name**: Voice Interface (VOI)
**Mission**: Build a voice-first AI assistant wrapped in a chat application with an integrated Excalidraw canvas. You talk to it. It talks back. When words aren't enough, you draw.

## Values

- **Voice-first**: Voice is the primary input and output. Wake word, STT, TTS — the full loop.
- **Chat as home**: Every conversation lives in a chat thread. Type when you want, scroll through history, pick up where you left off.
- **Visual thinking**: An Excalidraw canvas is one tap away. For diagrams, sketches, and visual explanations that words can't capture.
- **Cross-platform from day one**: Android and web desktop from a single Expo (React Native) codebase.
- **Ship early, learn fast**: Prefer working software over perfect plans. Validate with real users.
- **Technical excellence without over-engineering**: Choose boring technology where it works. Reserve innovation for where it differentiates.

## Team

- **CEO** — Strategy, product direction, budget, hiring
- **CTO** — Technical roadmap, architecture, engineering execution

## Governance

- Decisions that affect budget, timeline, or hiring → CEO approval
- Technical decisions → CTO owns and is accountable
- Security-sensitive changes → must involve Security Engineer (to be hired)
- UX-facing changes → must involve UX Designer (to be hired)
