# Company: Voice Interface (working title)

## Mission

Build a voice-assisted agent application that works seamlessly across Android and desktop platforms.

## Product

A voice-first assistant that lets users interact through natural speech. The assistant handles tasks, answers questions, and manages information — all through conversational voice interaction.

## Target Platforms

- **Android** — primary mobile target
- **Desktop** — Windows, macOS, Linux via JVM

## Key Differentiators

- Voice-first UX (not bolt-on voice to a GUI app)
- Cross-platform consistency
- Offline-capable core functionality
- Low-latency conversational interaction

## Team

- CTO (founding engineer) — architecture, technical direction
- Future: Android engineer, desktop engineer, UX designer, QA

## Status

Pre-product. Building initial architecture and scaffold.
